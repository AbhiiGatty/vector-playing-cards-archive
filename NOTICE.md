This repository is an **archive mirror** of the `vector-playing-cards`
project that was originally hosted on the Google Code Project Hosting
Service (which was turned down in early 2016). It is preserved here
because the original Google Code archive does not always make the files
trivially fetchable from a CDN.

## What is preserved here

All four published version bundles (1.0, 1.1, 1.2, 1.3) of both the
SVG and PNG card sets, organized by version under `svg/` and `png/`.
Filenames are unchanged from the upstream zip archives.

This repository adds:

- A README explaining the structure and how to use the assets via a
  CDN such as jsDelivr.
- A LICENSE file with the LGPL-3.0 text.
- This NOTICE file documenting the source and attribution chain.
